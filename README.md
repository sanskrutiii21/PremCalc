# ✨ Premium Glassmorphism Calculator

A stylish and fully functional calculator developed using HTML, CSS, and JavaScript.
Designed with a modern glassmorphism interface, animated backgrounds, and smooth user interactions to deliver a premium calculator experience.

## 🚀 Highlights

* Elegant Premium Design
* Responsive for Mobile & Desktop
* Animated Gradient Background
* Smooth Button Hover Effects
* Real-Time Calculations
* Keyboard Input Support
* Clean UI with Glassmorphism Style
* Fast and Lightweight

## 🧮 Supported Operations

* Addition
* Subtraction
* Multiplication
* Division
* Percentage
* Decimal Calculations
* Clear & Delete Functions
